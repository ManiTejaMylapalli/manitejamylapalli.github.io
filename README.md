# Customizing Your Portfolio

Welcome to your portfolio! This guide will help you customize your portfolio according to your preferences.

## Steps to Customize:

1. **Clone the Repository**:
   Open your terminal and run the following command:
   ```bash
   git clone https://github.com/ManiTejaMylapalli/manitejamylapalli.github.io.git
   ```

2. **Navigate to the Directory**:
   ```bash
   cd manitejamylapalli.github.io
   ```

3. **Modify the Content**:
   - Open the `index.html` file to change the main content.
   - Add your projects, skills, and contact information.

4. **Update Styles**:
   The CSS file is located in the `css` folder. Edit the `styles.css` file to change the appearance of your portfolio.

5. **Preview Changes**:
   Open the `index.html` file in your browser to see your changes live.

6. **Commit and Push Changes**:
   After you’re satisfied with your changes, use the following commands to save and push:
   ```bash
   git add .
   git commit -m "Customized portfolio" 
   git push origin main
   ```

7. **Deploy Changes**:
   Your portfolio will be updated automatically since it's hosted on GitHub Pages.

## Additional Tips:
- Feel free to explore different layout options and colors to make it uniquely yours!
- Don’t forget to check for responsiveness on different devices.

Happy customizing!
